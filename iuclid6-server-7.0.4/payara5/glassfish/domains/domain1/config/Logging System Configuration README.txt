By default, the level of detail of the logging is set to full debug.
To reduce the amount of disk space taken up by log files, configure the logging to record only what you need, by editing the file logback.xml.
The settings are described in the manual on the IUCLID website at:
https://iuclid6.echa.europa.eu/documents/1387205/1506740/installation_manual_server_en.pdf
